# BRC Crane Services Website

A modern, responsive marketing site for a crane company built with Next.js 14 and Tailwind CSS.

## Develop

```bash
npm install
npm run dev
```

Runs on http://localhost:3001

## Build

```bash
npm run build
npm start
```

## Features
- Responsive design, mobile-first
- Sections: Hero, Services, Fleet, About, Contact CTA
- Accessible components and semantic HTML
- Simple inline SVG illustrations (no external assets required)
